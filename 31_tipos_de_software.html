3.1. Tipos de software

Los programas y aplicaciones que se usan en los ordenadores y dispositivos que utilizamos a diario no son todos iguales. Hay software libre, comercial, con derechos de autor, modificable o redistribuible. A continuación vamos a estudiar la clasificación de los principales tipos de software según las licencias que los regulan:

  • Software propietario

El software propietario pertenece exclusivamente a una persona o empresa y el código fuente con el que está realizado solo puede verlo o modificarlo su dueño. No siempre se cobra por el uso de un software propietario (por ejemplo, muchas de las apps son programas gratuitos que se pueden instalar libremente). Sin embargo las cosas no siempre son como parecen y lo normal es que, o bien estas aplicaciones se nutran económicamente a partir de la publicidad incluida en su interior, o bien obtienen algún otro tipo de contraprestación de las que muchas veces ni siquiera somos conscientes, como pueden ser nuestros hábitos de consumo, amistades, contactos y un largo etcétera que podemos resumir pensando en que lo que estamos cediendo son nuestros datos personales.

  • Software comercial

Es el programa que una empresa comercializa con ánimo de lucro. Todo el software comercial es propietario pero esto no sucede al revés. Es decir, un software propietario no tiene que ser comercial . Ejemplos: Microsoft Office, Windows 10. En este tipo de software el código fuente pertenece al desarrollador y no se da acceso a terceros. Por tanto, los únicos que pueden modificar el código son los propios creadores.

  • Freeware

Es software gratuito pero no libre. Es también software propietario. Es un tipo de programas que se distribuye sin coste, disponible para su uso, pero que mantiene restricciones en su copyright. Por tanto, no se puede modificar, vender o usar libremente como ocurre con el software libre. Por ejemplo, Avast, Adobe Flash Player.

  • Software libre

Se puede usar, copiar, modificar y redistribuir libremente. Su código fuente está disponible, lo que se conoce como código abierto. Este código puede ser estudiado, modificado y usado libremente con cualquier finalidad y redistribuido con cambios o mejoras sobre él. Ejemplos: LibreOffice, GIMP, Audacity.
